Quick Start

Docker Run

docker run -d \
  -p 5173:5173 \
  -p 8000:8000 \
  -v ./config.yml:/app/config.yml \
  ghcr.io/vibhuvioio/ldap-manager:latest

Docker Compose

version: '3.8'
services:
  ldap-manager:
    image: ghcr.io/vibhuvioio/ldap-manager:latest
    ports:
      - "5173:5173"
      - "8000:8000"
    volumes:
      - ./config.yml:/app/config.yml

Access the UI at http://localhost:5173
