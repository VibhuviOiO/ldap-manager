## What Was Implemented

### Backend (API)

**File:** `backend/app/api/backup.py` (450+ lines)

**Endpoints:**

1. **`GET /api/backup/export/{cluster}`**
   - Exports entire LDAP directory to LDIF format
   - Similar to `slapcat -b "dc=example,dc=com"`
   - Returns downloadable `.ldif` file
   - Filename: `{cluster}-{timestamp}.ldif`

2. **`GET /api/backup/status/{cluster}`**
   - Checks if backup is available for the user
   - Returns availability status and reason

**Features:**
- ✅ Complete directory export (all entries + all attributes)
- ✅ RFC 2849 compliant LDIF format
- ✅ Base64 encoding for binary/non-ASCII data
- ✅ Operational attributes included (createTimestamp, etc.)
- ✅ Proper LDIF headers with metadata
- ✅ Access control enforcement
- ✅ Comprehensive error handling
- ✅ Audit logging with username

**Access Control:**
- **Keycloak mode:** Admin role required
- **Read-write mode:** Available to all users
- **Read-only mode:** Not available (403 Forbidden)

## Usage Examples

### Via Web UI

1. Login as admin (Keycloak mode) or access app (read-write mode)
2. Navigate to cluster details page
3. Click "Backup" button in toolbar
4. File downloads: `vibhuvioio.com-20260130-120530.ldif`

### Via API

**Check availability:**

```bash
curl -H "Authorization: Bearer $TOKEN" \
  http://localhost:8000/api/backup/status/vibhuvioio.com | jq
```

**Export backup:**

```bash
curl -H "Authorization: Bearer $TOKEN" \
  -o backup.ldif \
  http://localhost:8000/api/backup/export/vibhuvioio.com
```

### Restore from Backup

**Using ldapadd:**

```bash
ldapadd -x -D "cn=Manager,dc=vibhuvioio,dc=com" \
  -W -c -f vibhuvioio.com-20260130-120530.ldif
```

**Using slapadd (offline):**

```bash
sudo systemctl stop slapd
sudo slapadd -l vibhuvioio.com-20260130-120530.ldif
sudo systemctl start slapd
```

## Automated Backup Script

**Example cron job:**

```bash
#!/bin/bash
# /opt/backup-ldap.sh

CLUSTER="vibhuvioio.com"
BACKUP_DIR="/backup/ldap"
DATE=$(date +%Y%m%d)
TOKEN="your-admin-token"

mkdir -p $BACKUP_DIR

# Export backup
curl -H "Authorization: Bearer $TOKEN" \
  -o $BACKUP_DIR/$CLUSTER-$DATE.ldif \
  http://localhost:8000/api/backup/export/$CLUSTER

# Compress
gzip $BACKUP_DIR/$CLUSTER-$DATE.ldif

# Retention (30 days)
find $BACKUP_DIR -name "*.ldif.gz" -mtime +30 -delete

echo "Backup completed: $CLUSTER-$DATE.ldif.gz"
```

**Crontab:**

```bash
# Daily at 2 AM
0 2 * * * /opt/backup-ldap.sh >> /var/log/ldap-backup.log 2>&1
```

- Secure file permissions (chmod 600)
- Off-site storage
- Regular backup rotation
